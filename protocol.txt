This document details the protocol for interacting with the NewBank server.  

A customer enters the command below and sees the messages returned 

SHOWMYACCOUNTS
Returns a list of all the customers accounts along with their current balance 
e.g. Main: 1000.0 

NEWACCOUNT <Name>
e.g. NEWACCOUNT Savings
Returns SUCCESS or FAIL

MOVE <Amount> <From> <To>
e.g. MOVE 100 Main Savings 
Returns SUCCESS or FAIL

SHOWCURRENTBALANCE <Account Type>
e.g. SHOWCURRENTBALANCE Checking
Can also type ALL to get current balance for all acc types
Returns Account type + current balance

MAKEAPAYMENT <Account type> <Payee> <Payee Account> <Ammount>
e.g. MAKEAPAYMENT John Checking Bhagy Main 100
Returns Account paid from + it's new balance + payee name + account + its new balance

UPDATEPASSWORD <New password>
e.g. UPDATEPASSWORD letmein
Sets password to 'letmein', returns FAIL if password is insufficiently complex

END
Logs out and closes connection to a client
